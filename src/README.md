## Roadmap

```
📁 lexer            -=-   Recognising the symbols in the text
📁 parser           -=-   Parsing the symbols to Abstract Syntax Tree
📁 custom           -=-   Things like in-lang syntax errors, token definitions, etc.
📝 math-lang.ts     -=-   Exposing API for using the language
📝 interactive.ts   -=-   Simple implementation of API in math-lang.ts
```
